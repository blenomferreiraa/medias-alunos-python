# 📚 Sistema de Média de Alunos em Python

## 🚀 Sobre o Projeto

Este foi um dos projetos que desenvolvi durante meus estudos de Python com foco em lógica de programação e estruturas básicas da linguagem.

O objetivo do sistema é:

* cadastrar vários alunos;
* receber 4 notas para cada aluno;
* calcular a média;
* informar a situação final do aluno:

  * ✅ Aprovado
  * ⚠️ Recuperação
  * ❌ Reprovado

Além disso, utilizei a biblioteca **Colorama** para deixar o terminal mais visual e organizado com cores.

---

# 🧠 Conceitos praticados

Neste projeto pratiquei:

* `for`
* listas
* entrada de dados com `input`
* manipulação de listas com `append`
* cálculo de média
* condicionais (`if`, `elif`, `else`)
* formatação de strings (`f-string`)
* biblioteca externa (`colorama`)
* repetição baseada na quantidade informada pelo usuário

---

# 📌 Funcionamento do Sistema

O usuário:

1. informa a quantidade de alunos;
2. digita o nome de cada aluno;
3. informa 4 notas;
4. o sistema calcula automaticamente a média;
5. exibe a situação final do aluno.
